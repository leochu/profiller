Developer's Guide

1. Create the MD5 Hash

Be sure you trim and change all charactesr to lower case. e.g. CryptoUtils.md5("profiller@example.com").toLowerCase(); MD5 Hash: c29f93b9ad8abd263b5b832d265dee85

2. Invoke REST API with MD5 Hash

http://profiller.heroku.com/api/users/MD5Hash

3. Parse JSON response

Payload: { email: "profiller@example.com", firstName: null, lastName: null, emailMD5: "c29f93b9ad8abd263b5b832d265dee85" }
